# Assets

contains the element used in the game:

- animations : videos, sprites, animations and animator controllers 
- Images: different sprites and backgrounds
- Materials: unused atm
- Nytrock: library used for the font of texts
- Prefabs: stores unity prefabs 
- Scenes: contains the different scenes of the game 
- Scripts: contains the scripts (c# files)
- Settings: basic unity2D project settings
- Sounds: music of the game
- TextMesh Pro: library used for the dialogues
