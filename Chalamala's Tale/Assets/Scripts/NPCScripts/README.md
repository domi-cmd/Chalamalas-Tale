The dialogues are stored in an Unity database called DB_dialogues, it is possible to open it with the inspector in Unity to add new lines (using the + sign).
Each text bubble is shown as $[speaker]$: text. The scene is used to assign the bubbles to the corresponding dialogue file 


![image](../../READMEimages/DB_dialogues_inspector.png)


One the lines are written, you can set it to the dialogue element by adding the DB and using the desired scene_id


![image](../../READMEimages/dialogue_setup.png)
